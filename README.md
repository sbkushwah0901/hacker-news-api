# Hacker News Viewer

This project is a web application that allows users to view the newest stories from Hacker News using a front-end Angular app that communicates with a back-end RESTful API built with C# .NET Core.

## Features

 * View the newest stories from Hacker News.
 * Caching of the newest stories to improve performance.
 * Automated tests for the back-end API.

## Technologies Used

 * Angular (Front-end)
 * C# .NET Core (Back-end)
 * Hacker News API

## Getting Started

## Prerequisites

 * Node.js and npm installed to run Angular.
 * Visual Studio or Visual Studio Code with .NET Core SDK installed.

## Installation

  1. Clone the repository:
    * git clone https://github.com/yourusername/hacker-news-viewer.git
    * cd hacker-news-viewer

## Caching
The back-end API uses in-memory caching to store the retrieved stories temporarily for improved performance.

## Automated Tests
Automated tests are implemented for the back-end API to ensure code quality and functionality.
